
# Polar Arcondicionados

Projeto desenvolvido para Engenharia de Softwre.


## Deploy

Para o deploy:

```bash
  npm install
  
  npm run build

  npm run dev

  php artisan storage:link
```


## Stack utilizada

**Front-end:** JS, VITE, SCSS

**Back-end:** Node, PHP
