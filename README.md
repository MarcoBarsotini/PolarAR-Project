
# Polar Arcondicionados

Projeto desenvolvido para Engenharia de Softwre.
(Para ganhar acesso admin, mude a coluna chamada USER_ISADMIN de false para TRUE.)

## Deploy

Para o deploy:

```bash
  npm install
  
  npm run build

  npm run dev

  php artisan storage:link

  php artisan db:seed
```


## Stack utilizada

**Front-end:** JS, VITE, SCSS

**Back-end:** Node, PHP
